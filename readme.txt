Super simple Twitter Widget for listing hashtag matched tweets on your site.

== Description ==
Add the TwitterBox widget to your zones and set header and hashtags.

== Installation ==
1. Upload 'twitterbox' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to your themes widget settings in wp-admin
4. Add the widget "TwitterBox" to any zone
5. Name some hashtags with commas between the words and save
6. Look at your widget/sidebar pages and TwitterBox lists tweets containing matches for given hastags
